# Quiz
 Vuejs Application
#Vue using CLI
1. npm install -g@vue/cli 
(If there is old version uninstall and reinstall the vue-cli)
2. npm uninstall -g vue-cli
3. npm install -g @vue/cli
4. vue --version
#create a application
1. vue create appName (Select Default and use npm)
2. cd appName
3. npm run serve
4. npm run build
